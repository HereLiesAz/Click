# Screens

This document provides a breakdown of all screens in the Click application.

## 1. Main Screen (`MainActivity`)

The main screen is the primary UI of the app. It allows the user to:

*   Enable/disable the different trigger features.
*   Adjust the sensitivity of the "Tap" trigger.
*   Grant the necessary permissions.

## 2. Tap Calibration Screen (`TapCalibrationActivity`)

The tap calibration screen allows the user to calibrate the sensitivity of the "Tap" trigger.

## 3. Gesture Capture Screen (`GestureCaptureActivity`)

The gesture capture screen allows the user to record the on-screen location of the camera shutter button.
